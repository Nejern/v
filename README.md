# V

My [Neovim](https://neovim.io/) config

## Install

> Unix, Linux

1. Clone a repo to config folder
	```
	cd ~/.config
	git clone https://github.com/Nejern/v nvim
	```

2. In Neovim command mode write
	```
	PackInstall
	PackCompile
	```
