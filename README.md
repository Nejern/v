# v

My [Neovim](https://neovim.io/) config

## Install

> Unix, Linux

1. Clone a repo to config folder

	```
	cd ~/.config
	git clone https://github.com/Nejern/v nvim
	```

2. Install [Packer.nvim](https://github.com/wbthomason/packer.nvim)

3. In Neovim command mode write

	```
	PackInstall
	PackCompile
	```
